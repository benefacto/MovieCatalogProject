Prerequisites: 
-- Visual Studio
-- A modern web browser

To run the application, perform the following steps:
1. Clone or download project from Github
2. Open MovieCatalog.sln file in Visual Studio
3. Select top menu "Build" and click "Rebuild Solution"
4. Wait for rebuild to be confirmed as successful in bottom output window
5. Select top menu "Debug" and click "Start Without Debugging"
6. Navigate to http://localhost/app/ in a modern web browser:
i.e. Google Chrome, Mozilla Firefox, or Microsoft Edge
